# Vibe Georgia SEO Top 3 Roadmap

ეს ფოლდერი არის `vibegeorgia.com`-ის SEO განვითარების სამუშაო დოკუმენტი.

მიზანი:
- Google-ში ძლიერი ორგანული ზრდა
- მაღალი intent-ის ტურისტულ keyword-ებზე top 3 პოზიციებისთვის ბრძოლა
- ტექნიკური SEO, content SEO, internal linking და conversion trust signals-ის გაძლიერება

ფაილები:
- `01-current-audit.md` — ამჟამინდელი მდგომარეობა და მთავარი ხარვეზები
- `02-priority-roadmap.md` — პრიორიტეტებით დალაგებული სამუშაო გეგმა
- `03-keyword-cluster-plan.md` — რომელი თემები და გვერდები გაკლია რეალური ზრდისთვის
- `04-content-briefs.md` — ახალი landing page-ებისა და სტატიების brief-ები

მოკლე დასკვნა:
- ტექნიკური ბაზა უკვე წესრიგშია: canonical, sitemap, structured data, redirects, performance მნიშვნელოვნად გაუმჯობესებულია.
- ყველაზე დიდი დარჩენილი gap არის `SEO landing pages + keyword coverage + topical authority`.
- მხოლოდ ტექნიკური ოპტიმიზაციით top 3 მიღწევა რთულია.
- ყველაზე რეალური გზა არის:
  - კომერციული landing pages
  - destination cluster pages
  - ძლიერი blog support content
  - უფრო მკაფიო internal linking
  - მეტი trust/E-E-A-T სიგნალი
